Featured Project

NHL Analysis — 100 Years of Hockey Data

Data EngineeringMachine Learning VisualizationFull-Stack

A multi-phase capstone analyzing the evolution of the NHL across 100+ years of data — team performance, player statistics, fan engagement, and draft composition. Built across three progressive phases, each adding a new technical layer.

  • ETL Pipeline: Extracted NHL data via the sportsIO API, transformed with Pandas, and loaded into a PostgreSQL database using psycopg2 with composite primary keys for relational integrity.
  • Interactive Web App: Built with JavaScript, HTML/CSS, and Leaflet — featuring a player stat selector, an interactive team map with filterable layers (playoff %, Stanley Cup wins), and a draft nationality breakdown tool by year.
  • KMeans Clustering: Predicted playoff qualification using Goals For, Goals Against, and Goal Spread per game — revealing a ~3.5 goals/game threshold for playoff likelihood.
  • Logistic Regression: Optimized playoff predictor using team goal, penalty, and efficiency metrics.
  • Linear Regression: Forecasted individual player performance for upcoming seasons based on historical data trends.
  • AI Chatbot (Elodie): Built a conversational NHL assistant in Python for natural language queries against the dataset.
